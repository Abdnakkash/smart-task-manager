# Smart Task Manager

A responsive task management web app built with HTML, CSS, and Vanilla JavaScript.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed or pending
- Filter tasks by status
- Filter tasks by priority
- Search tasks by title or description
- Save tasks in LocalStorage
- Responsive dashboard layout

## Tech Stack

- HTML5
- CSS3
- JavaScript ES6
- LocalStorage

## What I Learned

This project helped me practice DOM manipulation, event delegation, array methods, localStorage, and building a full CRUD-style application with Vanilla JavaScript.
